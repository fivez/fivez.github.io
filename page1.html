Scheduling

Wake me up! permits to create a refined schedule for every one of your alarms.

Time and Name Click on the time displayed on the header to change the time (hour of the day and minutes) on which your alarm will ring.

Likewise, click on the label to change it.

Periodicity

No repetition

If the alarm is set as "do not repeat", it will disable itself when all schedules dates have occured.

Multiple days can be selected. By doing so, the alarm will ring on the selected days, and only then disable itself.
The selected dates will be the one immediately following the date on which the alarm is set.

By not selecting any days, the alarm will ring on the next possible date, possibly on the same day, otherwise, on the day after. Then, it will disable itself. Enabling it again will apply the same rule.

Repeat : default type

By clicking on "do not repeat", you can set the alarm to repeat every week. It will then display "Repeat".

With the "Repeat" option, at least one day has to be selected.

By not selecting any days, the alarm will ring on the next possible date, possibly on the same day, otherwise, on the day after. Then, it will disable itself. Enabling it again will schedule it again the same way.

Repeat : until a date

By selecting the option "Until", you can define a date on which the alarm will then disable itself.

Repeat : every X weeks

You can also change the frequency of the repeat option. You can schedule it for every week, every two weeks, up to every eight weeks.

Gentle Wake Up

By activating this option, the alarm will start ringing by a predefined number of minutes before its set up time (by default, 30 minutes).
Its ringtone will start quietly, and then grow louder and louder until the original set up time.
Likewise, its brightness will start at its lowest possible value, growing brighter every second.
For example, with default settings, if the alarm is set to ring at 07:00am, it will start ringing at 06:30am.
If the alarm is snoozed, it will ring again at its original time.

Exceptional times

With Exceptional Times, you can change, for one day, the time on which your alarm will ring.
It is compatible with "Gentle Wake Up".
Of course, you can define multiple Exception Times for one alarm!


  • FROM THE ALARM PARAMETERS - By using the "Exceptional times" card, from which you can add as many overrides as necessary, or delete the past ones.
  • FROM THE ALARM CARD - using the appropriate button, you can easily define an override for the next date on which the alarm is scheduled to ring!

Disabling an alarm

Automatically disabling your alarm based on your calendar

You can even synchronize those periods by synchronizing your alarm with your calendar!
Just select one calendar with which the alarm will synchronize, then your alarm won't ring for every day which has an all-day event set.
If you use all-day events for multiple purposes, you can also set Wake me up! to just take into account all-day events with a specific hashtag!

Define periods during which the alarm won't ring

Who never forgot to enable its alarm after a holiday?
With Wake me up!, you can prevent your alarm to ring without disabling it.
Just specify a period during which your alarm won't ring. It will start to ring again after this period.
Like with Exception Times, you can define multiple periods!

Skipping an alarm or going back to it

You can easily skip an alarm so that it won't ring on the next scheduled date, if you need to.
There are two ways to do it.


Using the notification

If the next alarm is scheduled to ring in more than two hours, the notification displays two buttons: "Previous" and "Next".

Note: the "Previous" button may not appear if there is no enabled alarm that could be schedule to ring in the future, but before the next scheduled alarm.


By using the "Next" button, the next scheduled alarm is skipped and the following one, if any, becomes the next scheduled alarm.

By using the "Previous" button, the alarm that was previously skipped (by using the "Next" button) is scheduled again.

Using the buttons on each card

For each enabled alarm, the "skip next" button permits to skip the next scheduled date for the alarm.
The "skip previous" button permits to go back to the previously skipped date and schedule it again.
