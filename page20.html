Night mode

A beautiful night mode for your nightstand.

Multiple ways of launching the Night mode are at your disposal.

  1. THROUGH THE APPLICATION - A button on the toolbar is dedicated to launching the Night Mode.Try it now!
  2. AS A SCREEN SAVER - Wake me up! can also be used as a Screen Saver. You just need to select the application in the dedicated Android Preferences. Try it now!
  3. WITH A WIDGET - Add the Night Mode widget on your homescreen to launch the Night Mode with one tap.
  4. USING TASKER - Launching the Night mode can also be done using a Tasker dedicated plugin. Doing so, you will be able to launch the Night mode the way you want.

Night mode options

Wake me up! comes with plenty of way to customize the Night mode.

  • CHANGE THE ACTION ON TOUCH - Three touch modes are available.
  • CURRENT WEATHER - The current weather situation can be displayed. An icon, a short summary and the current temperature will tell you what the weather is outside.Try it now!
  • BATTERY LEVEL - You'll also be able to see the battery level of your device. You can set the application to warn you if the Night mode has been launched with the device not being on charge.Try it now!
  • CUSTOMIZE THE FONT - Customize the font used to display the information on screen. You can change the font size and the font color.
  • CHANGE THE BACKGROUND COLOR- Customize the background color. Wake me up! cannot be held responsible for potential screen burns.
  • AUTOMATICALLY CLOSE - The Night mode can be set to automatically close when either a certain amount of time has elapsed, or when the battery level is below a user-defined thresold.
    Try it now!

Action on touch

  • QUIT - Like most Night modes on other applications, you can set the Night mode to always show you the information you've set it to display, and to quit on tap. What is displayed will move every minute, to prevent screen burn.Try it now!
  • MAKE BRIGHTER - Using this mode, the screen as a really low brightness by default. The information is always shown, like for the "Quit" mode. But, when tapping the screen, the screen becomes brighter for 10 seconds. A button is available to quit the Night mode.y w!
  • SHOW - Using this mode, the screen is entirely black. Information is shown when tapping on the screen, for 10 seconds. A button is available to quit the Night mode.Try it now!
