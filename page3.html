Skin customization

With Wake me up!, you can customize a lot of settings to make your alarms look unique.

Background Picture


From the phone

Click either on the "From the phone" checkbox or the icon to select a picture stored on your phone and use it as a background for your alarm.

If a picture has already been selected, click on the icon to select another one.

The header reflects the selected picture.

Weather related

You can choose to display a background picture in relation with the weather forecast.


This option works only if weather forecast is activated.
The picture is dynamic and not reflected in realtime in the edit screen header, but is reflected when testing the alarm.

Blurred Background

You can choose to also display a blurred version of the background picture on the information screens.

The information screens are:
- Weather forecast
- Calendar events

The information screens are displayed after an alarm has been dismissed.

This option works with both "From the phone" background and "Weather related" background.

Alarm Tint

It is possible to change the color associated to each alarm using this option.

A default set of colors is available. If a background picture, taken from the phone, is selected, colors are extracted from it and complement the default set.

The color will be used as a background in the alarm screen.

Text and buttons color

As for the alarm background, it is possible to change the color used to display texts and buttons on the alarm screen.

Colors can be from the selected picture or from a predefined set of colors.

If a picture is selected, colors are extracted from it. They are computed for readability, and matches the colors extracted from the background picture that can be used as Alarm Tint.
A color may seem to be displayed multiple times, but its transparency may differ.
